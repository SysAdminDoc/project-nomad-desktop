## 24. Understanding the 6 Services

NOMAD manages 6 separate tools. Each one is optional — install only what you need:

| Service | What It Does | Size | Who Needs It |
| --- | --- | --- | --- |
| **AI Chat** | Private AI assistant that answers questions offline | ~310 MB + models | Everyone — this is the core feature |
| **Information Library** | Offline Wikipedia, medical references, survival guides | ~60 MB + content packs | Everyone — essential reference material |
| **CyberChef** | Data encoding, encryption, decryption, hashing, file analysis | ~30 MB | Technical users who need encryption/encoding tools |
| **Education Platform** | Khan Academy courses, textbooks, progress tracking | ~200 MB + courses | Families with children, long-term learning |
| **Vector Search** | AI-powered document search for uploaded files | ~80 MB | Users who upload many documents for AI analysis |
| **PDF Tools** | Merge, split, compress, convert, OCR PDFs | ~100 MB + Java | Users who work with PDF documents frequently |

> **Minimal setup:** Install just AI Chat + Information Library for a powerful offline knowledge base. Add others as needed.
