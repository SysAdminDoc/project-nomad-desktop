## 39. AI Memory System

AI Memory lets you store persistent facts that the AI remembers across all conversations and copilot queries.

| Feature | Description |
| --- | --- |
| **Memory Panel** | Click the "Memory" button in the AI Chat header to view, add, or delete facts. |
| **Persistent Context** | Facts are injected into every AI interaction — chat, copilot quick-query, and SITREP generation. |
| **What to Store** | Location, group size, medical conditions, equipment specs, key concerns, ongoing situations, preferences. |
| **Emergency Sheet** | AI memory facts appear on the printable Emergency Reference Sheet under "Operator Notes." |

**Examples:** "Family of 4, two children ages 5 and 8", "Solar array is 5kW with 10kWh battery", "Well water requires UV treatment", "Bug-out location is the cabin at waypoint 'Safe House'".
