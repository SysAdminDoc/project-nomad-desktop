## 38. Task Scheduler

The Task Scheduler (Settings tab) lets you create recurring tasks for maintenance, medical, patrol, garden, and inventory activities.

| Feature | Description |
| --- | --- |
| **Recurring Tasks** | Set tasks to repeat daily, weekly, or monthly. When completed, the next due date auto-calculates. |
| **Categories** | Maintenance, Medical, Patrol, Garden, Inventory, Custom — color-coded in the task list. |
| **Assignment** | Assign tasks to team members from your Contacts list. |
| **Overdue Alerts** | Overdue tasks appear as predictive alerts in the alert bar with a "PREDICTED" badge. |
| **Sunrise/Sunset** | The dashboard sun widget shows sunrise, sunset, and day length for solar panel and patrol planning. |

**Examples:** "Check water filter" (weekly), "Generator oil change" (monthly), "Medication dose" (daily), "Patrol north fence" (daily).
