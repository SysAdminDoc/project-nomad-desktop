## 40. Printable Field Documents

NOMAD generates 9 printable documents for when screens aren't available. Access from the Home tab "Printable Field Documents" section or Settings tab.

| Document | Contents | Best For |
| --- | --- | --- |
| **Operations Binder** | Complete reference: TOC, contacts, frequencies, medical cards, inventory, checklists, waypoints, procedures, family plan | Keep in go-bag or command post. Replace monthly. |
| **Wallet Cards** | 5 credit-card-sized cards: ICE, blood type, medications, rally points, frequencies | Laminate and carry on person at all times. |
| **SOI** | Signal Operating Instructions: frequency assignments, call sign matrix, radio profiles, net schedule | Issue to all radio operators. Destroy when outdated. |
| **Emergency Sheet** | One-page critical data aggregate with contacts, supplies, patients, waypoints, weather, tasks, AI notes | Post on wall at base camp. Update weekly. |
| **Medical Cards** | Per-patient vital signs, medications, conditions | Give to each patient's caregiver. |
| **Bug-Out List** | Grab-and-go checklist with rally points | Post near exit doors. |
| **Inventory Report** | Full supply list with quantities, locations, expiration dates | Physical inventory verification. |
| **Contact Directory** | Complete personnel directory with all details | Keep copy at each rally point. |
| **Frequency Card** | Standard emergency frequencies + team contacts | Laminate pocket-sized for each radio operator. |
