## 16. Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| <kbd>Alt+1</kbd> through <kbd>Alt+9</kbd> | Switch between the primary workspaces |
| <kbd>Alt+T</kbd> | Open/close the timer widget |
| <kbd>Alt+C</kbd> | Open/close LAN chat |
| <kbd>Alt+N</kbd> | Create a new note |
| <kbd>Ctrl+K</kbd> | Open the command palette |
| <kbd>Escape</kbd> | Close any open overlay, panel, or dialog |
