## 41. Glossary

| Term | Definition |
| --- | --- |
| AI Model | A downloadable file that gives the AI its knowledge and reasoning ability. Larger models = better answers but more storage and RAM required. |
| BOB (Bug-Out Bag) | A pre-packed bag with essentials for a 72-hour evacuation. Also called a go-bag, GOOD bag, or INCH bag. |
| Burn Rate | How fast you are consuming a supply. Measured in daily usage. NOMAD uses this to calculate "Days Left." |
| CBRN | Chemical, Biological, Radiological, Nuclear. A category of threats requiring specialized protective measures. |
| Content Pack | A downloadable file containing offline reference material (Wikipedia articles, medical databases, survival guides, etc.). |
| GCS (Glasgow Coma Scale) | A 3-15 score measuring consciousness. 15 = fully alert, <9 = severe impairment, 3 = unresponsive. |
| GPX | GPS Exchange Format. A standard file format for sharing waypoints and tracks between GPS devices and mapping software. |
| hPa (hectopascal) | Unit of barometric pressure. Standard sea-level pressure is 1013 hPa. Falling pressure indicates approaching storms. |
| LAN | Local Area Network. Your home Wi-Fi network. NOMAD is accessible from any device on your LAN. |
| Meshtastic | An open-source project for long-range, low-power mesh networking using inexpensive radio hardware. No cell towers needed. |
| Node | A single NOMAD installation. When syncing between multiple computers, each is called a "node." |
| OPSEC | Operations Security. Practices to prevent adversaries from learning about your plans, capabilities, and vulnerabilities. |
| Rally Point | A pre-designated meeting location for your family or group during an emergency. |
| SITREP | Situation Report. A standardized format for summarizing the current state of an emergency or operation. |
| SOC (State of Charge) | How full a battery is, expressed as a percentage (0-100%). Similar to your phone battery percentage. |
| SpO2 | Blood oxygen saturation level. Normal is 95-100%. Below 90% is a medical emergency. |
| START Triage | Simple Triage and Rapid Treatment. A system for prioritizing multiple casualties: walk → breathe → pulse → mental status. |
| Waypoint | A saved location on the map with coordinates, name, and optional notes. |
