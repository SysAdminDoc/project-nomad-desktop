## 6. Offline Maps

Download maps for your region so you can navigate without cell service or internet. Select from 22 worldwide regions.

### 10 Map Tools

| Tool | What It Does |
| --- | --- |
| Drop Pin | Mark a point on the map with a label |
| Measure | Measure distance between two points |
| Save Waypoint | Save a named location to your database |
| Draw Zone | Draw a colored area on the map |
| Property Boundary | Draw a polygon and calculate area (acres) and perimeter (feet/miles) |
| Clear Pins | Remove all temporary markers |
| Print Layout | Generate a printable map page with title, compass, and scale |
| Bookmark | Save and recall specific map views |
| Bearing & Distance | Calculate direction (degrees + cardinal) and distance between two points |
| Export GPX | Export all saved waypoints to a GPS-compatible file |

> **Tip:** Enter coordinates directly (like "38.8977, -77.0365") in the search box to jump to a specific location.
