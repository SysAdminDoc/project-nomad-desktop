About

Project N.O.M.A.D. for Windows — Native edition, no Docker required.

Based on Project N.O.M.A.D. by Crosstalk Solutions. Windows port by SysAdminDoc.

Manages services as native processes instead of containers. Offline-first knowledge and AI command center.

Welcome to N.O.M.A.D.

Your offline survival command center.
